Sunne Processos
Visão Geral Dúvidas e Troubleshooting

Visão Geral do Ciclo Operacional

Este fluxograma detalha o ciclo de vida de um negócio e ticket dentro do HubSpot da Sunne, desde a validação comercial até o rateio concluído, incluindo as interações com o n8n e outras ferramentas.

graph TD
    A[Início: Deal em "Para Validação"] --> B{Decisão: existe_pendencia_para_validacao_comercial?};
    B -- "false (Aprovado)" --> C[WF2: Validados -> Fechado];
    B -- "true (Pendência)" --> D[WF3: Pendências ou Cancelamento];
    B -- "vazio (Revalidação)" --> E[WF4: Revalidação após Correção];

    C --> F[WF Bastão: Comercial -> CS];
    E --> A; /* Loop de revalidação */

    D -- "Closer corrige pendência" --> E; /* Ação manual do Closer */

    F --> G[WF Alocação: Movimentação para Cobrança];
    G --> H[Ação Manual: Cobrança e Geração do Ticket];

    H --> I[WF T1: Pagamento Confirmado -> Gerar Contratos];
    I --> J[n8n: Geração de Contratos];
    J --> K[WF T2: Gerar Contratos -> Subir no ClickSign];
    K --> L[n8n: Envio de Contratos ClickSign];
    L --> M[WF T3: ClickSign -> Aguardando Assinatura];
    M --> N[n8n: Checa se Deals foram Assinados];
    N --> O[WF T4: Assinado -> Baixando Documentos];
    O --> P[n8n: Baixar Documentos];
    P --> Q[WF T5: Drive OK -> Cadastro no OPS];
    Q --> R[n8n: Cadastra Beneficiários no OPS];
    R --> S[WF T6: OPS OK -> Aberto (Rateio)];
    S --> T[Ação Manual: Ticket para "Rateio Aprovado"];
    T --> U[WF7: Fluxo de Automações 7 - Rateio Concluído];
    U --> V[Fim: Deal em "Rateio Concluído"];
